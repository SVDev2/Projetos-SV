Projetos SV

🚀 Sobre o projeto

O Projetos SV é um site desenvolvido com Flask para reunir e apresentar projetos, serviços e criações desenvolvidos por SER VIVO (SVDev2).

O objetivo é manter um portfólio online onde seja possível compartilhar trabalhos de marcenaria, eletrônica, programação, mecânica, reparos, reciclagem, experimentos e diversos outros projetos, permitindo que qualquer pessoa possa conhecê-los de forma simples e organizada.
